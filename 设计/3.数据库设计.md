📐 设计部分 3/5：数据库设计（核心表结构）
数据库设计原则
所有表使用 InnoDB 引擎，字符集 utf8mb4
主键统一使用雪花算法生成的 BIGINT 类型 id
所有业务表包含审计字段：create_by / create_time / update_by / update_time / del_flag
枚举字段使用 TINYINT + 数据字典管理
金额字段使用 DECIMAL(12,2)
关键字段建立索引，外键关系通过应用层维护（不使用数据库外键）
